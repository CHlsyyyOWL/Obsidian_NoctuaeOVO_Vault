# 0 - Before depart
**对于在中国大陆办理学生签证的同学来说，学生签证一般路径如下：**

 * 请确保护照有效期6个月以上。如不在有效期内或者有效期不到6个月，需要先行更新护照。护照需要有至少两页以上的空白页。 
1.   先到法国高等教育署（CampusFrance）网站做预签证申请（procédure pré-consulaire）。申请账户，提交材料，缴费，参加面试评估（如有必要）。当法国高等教育署对材料处理完毕，申请人将会在个人页面收到一条留言，通知您材料已经转到了中智法签。具体信息：[https://www.chine.campusfrance.org/zh-hans/france-student-visa](https://eur01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.chine.campusfrance.org%2Fzh-hans%2Ffrance-student-visa&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504818037%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=4rBdub1SnjTUZdKHKBVB3SnMNdIJORLPtkwkRlVlabI%3D&reserved=0 "https://eur01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.chine.campusfrance.org%2Fzh-hans%2Ffrance-student-visa&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504818037%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=4rBdub1SnjTUZdKHKBVB3SnMNdIJORLPtkwkRlVlabI%3D&reserved=0")

      目前开放办理法国签证的城市有：北京，上海，广州，成都，沈阳，武汉，长沙，南京，昆明，杭州，济南，重庆，福州，深圳和西安。 
      
4.   在预约的日期前往签证受理中心提交签证申请，并请打印预约通知单。出席预约并在签证受理中心递交材料，缴费。 


一些提醒和常见问题：

 1.       CampusFrance要求的材料中，根据情况（已毕业，暂未毕业，国外文凭）会对申请人的文凭或进行学位认证网，或学信网，或经过国外文凭认证系统认证。这些手续都比较耗时间，尤其到了夏天认证高峰期。所以一定要留出足够的时间。 

2.       关于住宿证明材料。最理想的情况是大家在签证前找到合适的住宿，并获得一份住宿证明材料。如果大家一直没有找到合适的住处，并有可能要到巴黎后才能继续找住宿，这份住宿材料可以通过朋友的临时接待证明，酒店预订单等不同的材料来代替。大使馆的目的是要保证你们到了巴黎后有一个合适的落脚之地，不会流落街头。这份材料，据去年的经验，还没有特别的格式要求，如果今年还没有，大家准备的时候尽可能提供更多的信息，让大使馆觉得放心即可。为了给学生提供住房支持，巴政为大家提供了种类丰富的大学住宿以及私人住房的租房资源，相关信息可参考：[https://www.sciencespo.fr/students/en/living/housing](https://eur01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.sciencespo.fr%2Fstudents%2Fen%2Fliving%2Fhousing&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504926711%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=GlF8v%2Bq8DggUDeVW8GZWYxZ1KkDUihpp4ytaJ77w4tM%3D&reserved=0 "https://eur01.safelinks.protection.outlook.com/?url=https%3A%2F%2Fwww.sciencespo.fr%2Fstudents%2Fen%2Fliving%2Fhousing&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504926711%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=GlF8v%2Bq8DggUDeVW8GZWYxZ1KkDUihpp4ytaJ77w4tM%3D&reserved=0") 

3.     **学校官网在住宿的页面提醒大家不要在未见到房东和房子前交定金。我们也在此提醒大家尽量不要在这种不确定的情况下，有任何金钱来往，此前有同学有过不愉快的经历。****无论是在巴黎还是其他城市，房源肯定是有的，所有的同学最终都能找到相对满意的住宿，所以大家不用过于紧张，完全可以提前来法国，再集中看房找房。**  

4.      **如果各方面材料都已经准备好了，建议大家尽早开始申请。如果还没有完全准备好，也可以在CampusFrance网上先把账户开起来，根据要求先准备目前能够准备的材料。** 

对于在中国大陆以外的同学，请尽早与所在国和所在地法国大使馆以及Campusfrance联系，了解自己是否有资格在当地申请法国学生签证，如果可以在当地申请，签证程序如何。据我们以往的经验，不是全世界国家申请法国学生签证都是相同的步骤。**如需回国申请，则请尽早做好安排，留出足够的时间。** 
此外，巴政有一个专门为新生准备的网站，欢迎查看：[http://www.sciencespo.fr/students/en/get-started/international-students](https://eur01.safelinks.protection.outlook.com/?url=http%3A%2F%2Fwww.sciencespo.fr%2Fstudents%2Fen%2Fget-started%2Finternational-students&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504944058%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=pcU2yg7zjcmnRnZdfe7I0oAfKo9TaIP1PqIGZjAsNx0%3D&reserved=0 "https://eur01.safelinks.protection.outlook.com/?url=http%3A%2F%2Fwww.sciencespo.fr%2Fstudents%2Fen%2Fget-started%2Finternational-students&data=05%7C02%7Cjiatong.liu.23%40ucl.ac.uk%7Caf3593e46fcf4b6d73a908dd8d3b966a%7C1faf88fea9984c5b93c9210a11d9a5c2%7C0%7C0%7C638822013504944058%7CUnknown%7CTWFpbGZsb3d8eyJFbXB0eU1hcGkiOnRydWUsIlYiOiIwLjAuMDAwMCIsIlAiOiJXaW4zMiIsIkFOIjoiTWFpbCIsIldUIjoyfQ%3D%3D%7C60000%7C%7C%7C&sdata=pcU2yg7zjcmnRnZdfe7I0oAfKo9TaIP1PqIGZjAsNx0%3D&reserved=0")
